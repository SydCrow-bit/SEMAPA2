# SEMAPA IoT - Sistema de Gestión Comercial y Monitoreo Estratégico

Este es un sistema web robusto desarrollado en **Flask** diseñado para el control de infraestructura IoT, liquidación express de tarifas bajo el álgebra de bloques de SEMAPA y gestión comercial de contratos.

## 🚀 Tecnologías Utilizadas
- **Backend:** Python (Flask, Blueprints)
- **Base de Datos:** SQL (Conexión relacional optimizada para alta carga)
- **Frontend:** Bootstrap 5, Leaflet.js (Mapas interactivos de Cochabamba), Chart.js

## 🛠️ Instalación y Uso
1. Clonar el repositorio.
2. Crear un entorno virtual: `python -m venv venv`
3. Instalar dependencias: `pip install -r requirements.txt`
4. Configurar las variables de entorno en el `.env`.
5. Ejecutar con: `flask run` o `python app.py`
